# Digital Asset News Sources

แหล่งข่าว Digital Asset ที่น่าเชื่อถือสำหรับ Johny ใช้ research ทุกเช้า

## Primary Sources (ดึงทุกวัน)

| แหล่ง | URL | Focus |
|-------|-----|-------|
| CoinDesk | https://www.coindesk.com | ข่าวหลัก, market, regulation |
| CoinTelegraph | https://cointelegraph.com | narratives, DeFi, NFT, Web3 |
| The Block | https://www.theblock.co | institutional, on-chain data |

## สิ่งที่ต้อง extract จากแต่ละแหล่ง

1. **Trending asset/token** ที่พูดถึงมากที่สุดใน 24h
2. **Narrative หลัก** ของวัน (เช่น AI + crypto, RWA, Bitcoin halving)
3. **Market sentiment** (risk-on / risk-off / sideways)
4. **Thai market angle** ถ้ามี

## Fetch Order

1. CoinDesk → ถ้าสำเร็จ ใช้เป็น base
2. CoinTelegraph → เสริม narrative
3. Bitkub Blog → Thai context
4. ถ้าทุกแหล่งล้มเหลว → รายงาน "ดึงข้อมูลไม่ได้" อย่าแต่งข้อมูลเอง
